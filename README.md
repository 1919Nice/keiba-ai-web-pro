
# KEIBA AI WEB PRO V2

## 起動方法

```powershell
pip install -r requirements.txt
python app.py
```

ブラウザで開く:

```text
http://127.0.0.1:5000
```

## 反映済み

- レース名入力
- 公開出馬表URLから馬名・馬番などを取得する簡易取得器
- 取得失敗時のサンプル表示
- 穴狙い / バランス の2パターン予想
- 単勝 / 馬連 / 3連単
- 基本情報、枠順、斤量、馬齢、オッズ、血統、近走、競馬場適性、コース適性、騎手、調教師、展開、追い切り、パドック、返し馬、馬場、期待値
- 丞相風 / うまログ風の評価軸

## 注意

公開サイト解析はページ構造変更やアクセス制限で失敗する場合があります。
安定運用には正規データAPI、CSV連携、またはサーバー側取得処理の強化が必要です。
